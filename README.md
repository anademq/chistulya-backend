# Chistulya Backend API

Backend API на Laravel 13 + GraphQL (`rebing/graphql-laravel`) с JWT (`tymon/jwt-auth`) и refresh-сессиями.

## Ключевые директории

- `app/GraphQL/Queries` — query-операции
- `app/GraphQL/Mutations` — mutation-операции
- `app/GraphQL/Types` — GraphQL-типы
- `app/Services` — бизнес-логика домена
- `app/GraphQL/Middleware` — guard/gate логика
- `lang/ru` — локализация API/валидации/ошибок
